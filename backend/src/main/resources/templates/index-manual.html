Manual de Usuario para utilizar el Bloc de Notas

Para visualizar los datos de la base de datos de Usuarios con GET: http://localhost:8080/api/allUsers

Para visualizar los datos de la base de datos de Usuarios por ID con GET: http://localhost:8080/api/UserID

Para insertar datos a la base de datos de Usuarios con POST: http://localhost:8080/api/Insert

Para borrar datos de la base de datos de Usuarios con DELETE: http://localhost:8080/api/Delete
