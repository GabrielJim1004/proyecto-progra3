Manual de Usuario - API

Endpoint 1: Get Users

Endpoint: /user/all
Método: GET
Descripción: Llama a todos los usuarios de la base de datos.
Ejemplo: 

                    User{
                        UserID = int;
                        Name = String;
                        Lastnames = String;
                        Email = String;
                        Username = String;
                        Password = String;
                    }

Endpoint 2: User ID

Endpoint: /user/byid
Método: GET
Descripción: Llama a un usuario existente por ID de la base de datos.
Ejemplo: 

      
UserID(int)                                 
UserID(10)

Endpoint 3: Insertar Usuario

Endpoint: /user
Método: POST
Descripción: Inserta Usuarios a la base de datos.
Ejemplo: 

User{                                       
String name, String lastnames, String email, 
String username, String password}

Endpoint 4: Eliminar Usuario

Endpoint: /user
Método: DELETE
Descripción: Borra usuarios de la base de datos.
Ejemplo: 

DELETE Usuario by (ID)

Endpoint 5: Crear Nota

Endpoint: /note
Método: GET
Descripción:Crea una nueva nota.
Ejemplo: 

                    
note{
int noteID,
boolean status, 
String title,
String content, 
int userID
}

Endpoint 6: Ver Notas

Endpoint: /note/all
Método: GET
Descripción: Devuelve todas las notas.
Ejemplo: 

                    getAllNotes()

Endpoint 7: Actualizar nota por ID

Endpoint: /note/byid
Método: PUT
Descripción: Actualiza una nota existente por su ID
Ejemplo: 

                    updateNota(ID)

Endpoint 8: Nota Completada

Endpoint: /note/complete
Método: PUT
Descripción: Cambia el estado de una nota a completada por ID.
Ejemplo: 

                    updateNotaCompletada(true) by ID

Endpoint 9: Ver Notas

Endpoint: /note/complete
Método: GET
Descripción: Devuelve todas las notas Completadas.
Ejemplo: 

                    getAllNotes() -- status:true

Endpoint 10: Delete by ID

Endpoint: /note
Método: DELETE
Descripción: Elimina una nota por ID.
Ejemplo: 

                    Nota{int id}
